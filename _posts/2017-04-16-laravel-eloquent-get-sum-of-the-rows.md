最近写到相关的代码，直接上代码做个记录。

    public function scopeUserSum($query)
    {
        return $query->groupBy('username')->selectRaw('sum(data) as sum, username')->orderBy('sum', 'desc');
    }

如果只是纯粹求一列的和，在builder上直接用sum即可。
