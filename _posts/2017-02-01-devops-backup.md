![](https://cdn.kelu.org/blog/2017/02/crontab.jpg)

# ssh相关

ssh除了登录端口修改外，在当前用户的家目录下 ~/.ssh 添加 config 文件，用于快速登录其他服务器或 scp 进行文件传输。

    Host    tokyo
      HostName        xx.xx.xx.xx
      Port            1234
      User            kelu
      IdentityFile    ~/.ssh/xxx
    Host    fremont
      HostName        xx.xx.xx.xx
      Port            1234
      User            madcat
      IdentityFile    ~/.ssh/xxx
      
除此之外，还可以修改 /etc/hosts，用于快速ping某某网站等等。
      
      127.0.0.1       localhost
      xx.xx.xx.xx   tokyo
      xx.xx.xx.xx   aliyun
